Akses internet pada CMD/Terminal dengan proxy

  • Thu 12 December 2019
  • postingan
  • Rifky

Dalam berbagai kasus instalasi modul Python terkadang pengguna menghadapi masalah saat menggunakan akses jaringan dengan proxy. Untuk mengatasi hal tersebut perlu dilakukan pengaturan proxy pada CMD/Terminal

  1. Untuk sistem operasi Windows berikut adalah perintahnya
set HTTPS_PROXY=https://username:password@proxy:port
  1. Untuk sistem operasi MAC berikut adalah perintahnya
export proxy https_proxy=https://username:password@proxy:port
