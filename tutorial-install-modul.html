Tutorial Install Modul

  • Thu 12 December 2019
  • postingan
  • Rifky

Dalam berbagai kesempatan kita mungkin membutuhkan modul tambahan. Dalam tutorial ini akan digunakan pip.

Untuk memastikan bahwa pip telah terinstal jalankan perintah berikut pada CMD (Windows) atau Terminal (Linux/Mac)

pip --version

Apabila muncul error berarti pip belum terinstall. Apabila pip telah menunjukkan versinya maka proses instalasi dapat modul dapat dilakukan dengan perintah berikut

pip install nama_modul

Perintah di atas akan menginstall modul nama_modul secara online ke directory Python. Bagi yang menggunakan platform Anaconda saya menyarankan untuk menggunakan perintah berikut

conda install nama_modul

Perintah di atas akan menginstall modul pada directory Anaconda sehingga modul tersebut dapat diimport juga menggunakan Spyder. Instalasi menggunakan pip akan membuat Spyder tidak bisa mengakses
